# Анализ конференций TED

**Задача:** Разработать аналитический дашборд для анализа популярности тем, реакции аудитории, статистики выступлений и определения наиболее востребованных спикеров конференции.

## Реализация:
* Создан комплексный дашборд в DataLens с набором взаимосвязанных метрик для глубокого анализа конференций TED, включающий:
  * расчет ключевых метрик (количество конференций, выступлений, спикеров),
  * выявление топ-10 самых смешных и топ-20 самых популярных выступлений,
  * анализ профессиональной деятельности наиболее успешных спикеров,
  * детализацию по каждому выступлению
  * реализована система фильтрации для детального изучения по странам, конференциям, тегам и датам.

## Результат:
* созданный дашборд позволил систематизировать анализ конференций TED, что дает возможность оптимизировать организацию будущих конференций на основе данных.

## Инструменты:
`DataLens`, `SQL`
